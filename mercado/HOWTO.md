 
Executando aplicação
--------------------
Para executar é necessário um servidor de aplicação(ex: apache tomcat).

Para acessar a página principal do sistema, acesse: /mercado/index.html

Para adicionar um novo produto, acesse:   /mercado/novo-produto.html ou através do 
link "Adicionar novo produto" na própria página principal do sistema.


----------

Testes
------
Para testar a aplicação, primeiramente, inicie o servidor.
Após o servidor iniciado, siga o seguinte caminho até a classe de testes:
Pasta: src/test > 
pacote: mercado > 
classe: MercadoServiceTest.java
